# fable-pens-TheLastExperience
Fable port of https://codepen.io/ge1doot/pen/LkdOwj

# Try original version
```
> cd original
> open index.html
```

# To build and run

Read [fable documentation](http://fable.io/pages/getting-started.html)

And run

```
> dotnet restore
> yarn install
> cd pen/src
> dotnet fable yarn-run start
